RAG

This project is a simple implementation of a Retrieval Augmented Generation (RAG) system for enterprise use cases like document search and knowledge assistants. It allows users to upload PDF documents and ask questions, retrieving relevant content and generating accurate, context-aware responses.

Features
Supports multiple PDFs, context-based Q&A, vector search using FAISS, and conversational memory.

Tech Stack
Python, LangChain, OpenAI, FAISS, PyPDF

Learning Outcome
This project helped me understand how RAG systems combine retrieval and generation, and gave me hands-on experience in building real-world AI applications using LLMs and vector databases.
