Sales Insights Generator using Natural Language Queries

This project is a Natural Language to SQL system that allows users to query a sales database using plain English. The system converts user questions into SQL queries, executes them on a structured sales database, and generates meaningful business insights from the results.

It is built using Python, LangChain, and OpenAI, and demonstrates how AI can bridge the gap between natural language and structured data systems. The workflow includes three main steps: converting natural language into SQL, executing the query on a database, and generating human-readable insights from the results.

This project helps in understanding how AI-powered systems can simplify data analysis and make database interactions more intuitive for users without technical SQL knowledge.
